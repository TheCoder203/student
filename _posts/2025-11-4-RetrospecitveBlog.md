#### Kasm Setup

Open Terminal > Linux Commands

Useful commands and notes:

- mkdir, cd, ls
- Clone project: git clone https://_____
- Activate tools: Ruby, Python, Git SDLC
- Workflow: code -> make -> test -> commit

Useful commands (simple list):

- ls
- pwd
- mkdir
- cd
- git
- cat
- git clone
- git pull
- git commit
- git push

KASM setup commands (for first time set up):

```bash
mkdir opencs
cd opencs
git clone https://github.com/Open-Coding-Society/student.git
cd student/
./scripts/activate.sh  # prompts for Git UID and Personal Email
./scripts/venv.sh
code .  # Opens VS Code
```

Version check commands:

```bash
python --version
pip --version
ruby -v
bundle -v
gem --version
git config --global --list
```

Starting commands for each time you open Kasm:

```bash
cd opencs/student
source venv/bin/activate  # activate Python virtual environment
code .
```
